## 📋 项目介绍

源仓库是一个纯前端、开源、免费的订阅源管理平台，专注于为用户提供优质的开源阅读书源、影视仓订阅源和IPTV直播源。

### 🌟 核心优势

- **📚 开源阅读书源**：汇聚全网优质免费书源，支持各种阅读客户端
- **🎬 影视仓订阅源**：收集精选影视资源订阅源，提供高清观影体验
- **📺 IPTV直播源**：整理优质IPTV直播源，支持IPv4和IPv6
- **🔄 一站式管理**：统一管理、分享、获取各类订阅源
- **👤 用户友好**：简洁直观的界面，流畅的用户体验
- **🌓 深色模式**：支持深色模式切换，保护视力

## 🤝 贡献指南

### 提交优质订阅源

1. 访问项目GitHub仓库
2. 点击「Issues」→「New issue」
3. 选择「提交新源」模板
4. 填写源类型、源名称、订阅链接、源描述等信息
5. 点击「Submit new issue」提交
6. 我们会审核你的源，审核通过后会展示在平台上

## ❓ 常见问题

### Q: 如何添加新的书源或影视源？
A: 可以通过GitHub Issues的「提交新源」模板提交，或直接修改相应的JSON数据文件。

### Q: 为什么有些源无法使用？
A: 订阅源的可用性取决于源的提供者，我们会定期检查和更新源的状态。

### Q: 如何报告问题或提出建议？
A: 可以通过GitHub Issues提交问题或建议，我们会及时处理。

### Q: 如何使用一键导入功能？
A: 在系统浏览器或开源阅读客户端中，点击书源卡片上的「一键导入」按钮，系统会自动调起客户端并导入书源。
